rebase对比merge
优点：

  • 合并后的结果很清晰，只有一条线

缺点：

  • 如果一旦出现冲突，解决冲突很麻烦，可能要解决多个冲突，但是merge出现冲突只需要解决一次

使用rebase应该在需要被rebase的分支上操作，并且该分支是本地分支。
